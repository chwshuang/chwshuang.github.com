1.2 原因

这类异常一般是文章中使用了now( ), 小括号( )属于特殊字符,在编译文章时将now( )当函数处理了,结果找不到函数,就报错了。

1.3 解决方案

可以参考Markdown语法第(0)章-特殊字符
将now( )的小括号通过

3.2 原因

参数 说明
repo OpenShift 库(Repository)地址
message 自定提交信息 (默认为 Site updated: { { now(’YYYY-MM-DD HH:mm:ss’) } })

可以参考Markdown语法第(0)章-特殊字符
